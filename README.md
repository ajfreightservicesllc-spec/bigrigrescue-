# Big Rig Rescue

Emergency truck, trailer, diesel & RV repair directory along Interstate 40.

## Site Details

- **Live URL:** https://bigrigrescue.web.app
- **Custom Domain:** www.bigrigrescue.co
- **GA Tag:** G-RBZVCTXC7R
- **Firebase Project:** aiansweragency-main
- **Firebase Hosting Site:** bigrigrescue

## Features

- Emergency repair shop search across 8 states (CA, AZ, NM, TX, OK, AR, TN, NC)
- Location-based search with geolocation
- Service filters (roadside, 24/7, truck specialist, RV, reefer, tire, engine, trailer, brakes, electrical)
- Sort by distance, rating, reviews, or name
- Direct call, map, and website links for each shop
- Responsive design with dark orange/red gradient theme
- Animated hero with honking truck
- 2,159 shop pages generated

## Shop Data

- 664 total emergency repair shops across I-40 corridor
- Real phone numbers and website links
- Star ratings and review counts
- Service specialties tagged for each location
- City-specific pages for 136+ cities

## Development

- Single-page + multi-page static HTML site

- Firebase Hosting deployment

firebase deploy -c firebase_bigrig.json --only hosting:bigrigrescue --project aiansweragency-main
